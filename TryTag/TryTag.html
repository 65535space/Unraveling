HTML
HyperText Markup Languageの略称で、ウェブページの構造や内容を記述する言語です。
CSS
Cascading Style Sheetsの略称で、ウェブページの見た目やレイアウトを記述する言語です。
JavaScript
ウェブページに動的な機能やインタラクションを追加するためのプログラミング言語です。
